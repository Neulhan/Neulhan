
## install dependencies
```shell
pnpm i
```

## run dev
```shell
pnpm run dev
```

## publish to gh-pages
```shell
pnpm run deploy
```


## to use S3 copy paste image on md file
1. Install VsCode Extension `mushan.vscode-paste-image`
2. brew install
    ```shell
    brew install fswatch awscli
    ```

3. aws configure
    ```shell
    aws configure
    ```
